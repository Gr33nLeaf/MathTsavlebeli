# mathწავლებელი (MathTsavlebeli)
## 🚀 ფუნქციონალი
პროექტი „mathწავლებელი“ შეიქმნა აბიტურიენტებისთვის, მათემატიკის სასწავლო რესურსების გაფანტულობისა და კერძო რეპეტიტორთან მომზადებით გამოწვეული პრობლემების გადასაჭრელად. პროდუქტი აერთიანებს 2005 წლიდან არსებულ საგამოცდო არქივს, ინტერაქტიულ სახელმძღვანელოებს დაფარული პასუხებით, ონლაინ იმიტირებულ გამოცდებსა და ჩატბოტს — „AI რეპეტიტორს“. 
თანამედროვე ტექნოლოგიების (React, Gemini API) გამოყენებით აგებული ეს პლატფორმა პასუხობს განათლების გაციფრულების აქტუალურ გამოწვევებს და მიზნად ისახავს მაღალი ხარისხის მათემატიკური განათლების დემოკრატიზაციას, მომხმარებლის ფინანსური მდგომარეობის მიუხედავად, მისთვის ხელსაყრელ დროს.

ვებ-აპლიკაციის ოთხი ძირითადი მოდულია:
* ტესტების არქივი: 2005-2024 წლების ყველა ვარიანტი ციფრული წამკითხველით.
* სამეცადინო მასალები: ინტერაქტიული წიგნები დაფარული პასუხებით თვითანალიზისთვის.
* იმიტირებული გამოცდა: ონლაინ სივრცე ტაიმერით და მყისიერი შეფასებით.
* AI მასწავლებელი: ჩაშენებული ჩათ-ბოტი დეტალური ახსნა-განმარტებებისთვის.


### 🛠 გამოყენებული ტექნოლოგიები
ტექნიკური სრულყოფილებისთვის გამოვიყენეთ უახლესი ტექნოლოგიები:
* **React.js** — სწრაფქმედი ინტერფეისისთვის.
* **Tailwind CSS** — მობილურ მოწყობილობებზე სრული ადაპტაციისთვის.
* **Google Gemini API** — ეს არის "ტვინი", რომელიც ამუშავებს AI მასწავლებელს და პასუხობს ქართულ ენაზე.
* **Runtime** — [Node.js](https://nodejs.org/)
* **Framework:** — [React 19](https://react.dev/)
* **Language:** — [TypeScript](https://www.typescriptlang.org/)
* **Build Tool:** — [Vite](https://vitejs.dev/)
* **Styling:** — [Tailwind CSS](https://tailwindcss.com/)
* **AI Integration:** — [Google GenAI SDK](https://www.npmjs.com/package/@google/genai)

## 📂 პროექტის სტრუქტურა

Based on the configuration files:

*   `index.html`: მთავარი ფაილი, სადაც შემოტანილია ბიბლიოთეკები, სხვა ფაილებთან ერთად.
*   `src/`: სამომავლო ფაილი სადაც განთავსდება არა ფუნდამენტული ფაილები.
*   `package.json`: წევრები და სკრიპტები.
*   `metadata.json`: დამატებითი, მეტაინფორმაცია და აღწერა.

## 🏁 გამოყენება

პროექტის გასაშვებად მიჰყევით მომდევნო ნაბიჯებს.

### წინარე საჭიროების პროგრამული უზრუნველყოფა

*   Node.js (უახლესი ვერსია)
*   npm ან yarn

### Installation

1.  **გადმოიღე პროექტის ფაილები:**
    ```bash
    git clone https://github.com/your-username/mathtsavlebeli.git
    cd mathtsavlebeli
    ```

2.  **დააყენე მოთხოვნილებები:**
    ```bash
    npm install
    ```

3.  **შექმენი გარემო პროექტის გასაშვებად:**
    რახან ვიყენებთ `@google/genai`-ს, საჭიროა API Key-ს ქონა.
    *   შექმენი `.env` გაფართოების ფაილი მთავარ საქაღალდეში.
    *   დაამატე google api გასაღები (რომლის ცვლადის სახელიც სიტუაციურია, მაგ., `VITE_GOOGLE_API_KEY`).

4.  **გაუშვი დეველოპერული სერვერი**
    ```bash
    npm run dev
    ```

5.  **საბოლოოდ, ააწყე:**
    ```bash
    npm run build
    ```

## 📜 ბრძანებები

*   `npm run dev`: ქმნის Vite-ს დეველოპერის სერვერს.
*   `npm run build`: ქმნის საორიდუქციო აპლიკაციას.
*   `npm run preview`: Locally previews the production build.
